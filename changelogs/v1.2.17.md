# v1.2.17 - Encabezado horizontal en reglas

_Fecha: 2026-04-22_

## Resumen

Las tarjetas de reglas del laboratorio ahora muestran el tipo de regla y el titulo corto en una misma fila, para reducir la altura visual y dejar mas claro el bloque.

## Cambios clave

- El encabezado de cada tarjeta paso a un layout horizontal.
- El tipo de regla y el titulo corto ahora comparten la misma linea.
- El contenido de abajo se mantiene en una sola columna.
- La logica del laboratorio sigue igual; solo cambio la presentacion.

## Validaciones

- `npm run build`

## Notas

- Esta vuelta mejora la lectura rapida del panel sin tocar la semantica del sistema.
