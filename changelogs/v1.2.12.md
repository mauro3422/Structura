# v1.2.12 - N:N con tabla puente e inferencia de relaciones

_Fecha: 2026-04-22_

## Resumen

Esta iteracion amplia el laboratorio de tablas para detectar relaciones muchos a muchos a traves de una tabla puente, manteniendo el modelo pedagogico claro y sin depender de nombres visibles como fuente de verdad.

## Cambios clave

- Las relaciones dejaron de depender del nombre visible de la tabla y ahora usan ids estables.
- El panel de relaciones ahora puede inferir `N:N` cuando una tabla puente conecta dos tablas distintas mediante dos FKs.
- Se agrego una observacion especifica para identificar tablas puente candidatas.
- El panel de reglas del laboratorio explica cuando una relacion es directa, cuando se infiere y que queda para una fase futura.
- La cardinalidad sigue siendo didactica:
  - `1:N` se interpreta como maestra/detalle.
  - `1:1` sigue siendo una relacion a revisar si la FK no es estricta.
  - `N:N` se muestra como relacion derivada a partir de una tabla puente.
