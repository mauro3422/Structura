# v1.2.7 - Observaciones didacticas de tablas

## Resumen

Esta iteracion agrega un panel de observaciones para que el laboratorio de tablas avise errores y sugiera mejoras sin bloquear al usuario ni exigir configuracion extra.

## Cambios

- Se agrego un panel de observaciones con avisos e incidencias visibles.
- Se sugirieron nombres camelCase para tablas y columnas cuando el texto no sigue ese estilo.
- Se detectaron columnas duplicadas y tablas sin clave primaria.
- Se marco como relacion invalida la FK que no apunta a ninguna tabla.
- Se mantuvieron las alertas para cardinalidad 1:1 que necesita revision.

## Impacto

- El laboratorio se vuelve mas didactico y menos dependiente de configuracion manual.
- El usuario ve sugerencias concretas en vez de errores silenciosos.
- Las reglas de modelado quedan mas cerca de la teoria que se quiere ensenar.
