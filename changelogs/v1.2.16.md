# v1.2.16 - Reglas del laboratorio en una sola columna

_Fecha: 2026-04-22_

## Resumen

El panel de reglas del laboratorio de tablas se ajusta para que cada tarjeta mantenga sus elementos en una sola columna vertical. Esto evita el apretado visual que aparecia cuando los chips se repartian en dos columnas dentro de cada bloque.

## Cambios clave

- Cada tarjeta de reglas ahora apila sus elementos en una sola columna.
- Los chips internos dejaron de repartirse en dos columnas.
- El contenido se lee de arriba hacia abajo, como una lista corta por bloque.
- La logica del laboratorio sigue intacta; solo cambio la composicion visual.

## Validaciones

- `npm run build`

## Notas

- Esta vuelta apunta a legibilidad y no afecta el modelo ni las observaciones.
