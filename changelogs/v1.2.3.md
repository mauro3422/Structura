# v1.2.3 - Cobertura extra para lesson y widgets complejos

**Fecha de Lanzamiento:** 22 de Abril de 2026

## Resumen

Esta iteración amplió la red de seguridad automática del proyecto con tests sobre la página de lección, el renderer de secciones y dos widgets que todavía eran puntos sensibles: `magic-table` y `tree`. El objetivo fue validar render, estructura y una interacción mínima real sobre los bloques más complejos.

## Cambios Principales

- **Lesson coverage**: se agregó un test de `renderLesson` y otro de `renderLessonSection` para validar el wrapper de la lección y el fallback de secciones desconocidas.
- **Smoke de navegación**: se mantuvo y consolidó la prueba de arranque de la app con navegación a una ruta de lección.
- **Magic table**: se sumó cobertura del frame de render y de la evolución del bloque al hacer clic en un nodo.
- **Tree widget**: se validó el render del árbol y la expansión de nodos mediante la interacción del icono.
- **Ajustes menores de texto**: se limpiaron mensajes visibles en las vistas de lección para que la UI y los tests reflejen el mismo contenido.

## Impacto

- Aumentó la confianza para seguir tocando lecciones y widgets sin romper rutas críticas.
- Se cubrieron los componentes más dinámicos que quedaban fuera de la suite inicial.
- La base queda mejor preparada para agregar tests de regresión por bloque a medida que el sistema crece.
