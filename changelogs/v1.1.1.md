# v1.1.1 - Pulido visual de diagramas y comparación DNS

**Fecha de Lanzamiento:** 22 de Abril de 2026

## Resumen

Esta iteración se enfocó en la lectura visual de las lecciones de redes. Se simplificaron los diagramas Mermaid para que no se vieran comprimidos en mobile y se rearmó la comparación DNS como una tarjeta de flujo más clara, compacta y centrada.

## Cambios Principales

- **Diagramas más legibles**: los bloques Mermaid pasaron a usar un contenedor tipo canvas con scroll propio y límites de alto para conservar legibilidad sin aplastarse.
- **Comparación DNS simplificada**: la comparación entre nombre e IP se rediseñó como una sola tarjeta de flujo, con encabezado centrado y contenido menos pesado.
- **Ajuste de composición**: se afinó el espaciado, el ancho y la jerarquía visual del bloque de comparación para que encaje mejor con el resto de la lección.
- **Copias de contenido limpias**: se añadió el resumen explícito de la comparación en el módulo de redes para que la tarjeta tenga contexto desde el contenido.

## Impacto

- Las lecciones de redes quedan más limpias y fáciles de leer en pantallas chicas.
- La comparación DNS deja de parecer un bloque duplicado o excesivamente alto.
- El sistema conserva la misma arquitectura modular, pero la experiencia visual queda más compacta y clara.
