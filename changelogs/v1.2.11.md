# v1.2.11 - Clave compuesta y reglas mas claras

## Resumen

Esta iteracion agrega una pista visual para el caso de clave primaria compuesta y refuerza el mensaje pedagogico del laboratorio:

- Varias columnas marcadas como PK se interpretan como una clave compuesta informativa.
- El panel de reglas deja mas claro que lo obligatorio y lo sugerido no se mezclan.

## Detalles

- Se sumo una observacion de tipo info cuando una tabla usa mas de una columna como PK.
- El panel de reglas ahora explica que una tabla puede tener una clave compuesta.
- La validacion mantiene la prioridad de FK, destino y cardinalidad como verdad del modelo.
