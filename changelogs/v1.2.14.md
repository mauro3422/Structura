# v1.2.14 - Panel de reglas compactado en columnas

_Fecha: 2026-04-22_

## Resumen

El panel de reglas del laboratorio de tablas ahora usa una presentacion mas compacta y legible. En vez de bloques largos con listas verticales, cada regla se muestra como una columna con chips breves.

## Cambios clave

- Las descripciones del panel de reglas se acortaron para que ocupen menos alto.
- Las listas ahora se renderizan como chips en fila dentro de cada tarjeta.
- La grilla de reglas conserva las tres columnas, pero con una lectura mas liviana.
- El laboratorio sigue sin bloquear minúsculas o snake_case, solo sugiere mejoras cuando corresponde.

## Validaciones

- `npm run typecheck`
- `npm test`
- `npm run build`

## Notas

- Esta vuelta es puramente de pulido visual. La logica de relaciones y cardinalidad quedo intacta.
