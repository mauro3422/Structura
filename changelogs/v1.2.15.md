# v1.2.15 - Tarjetas de reglas mas compactas

_Fecha: 2026-04-22_

## Resumen

El panel de reglas del laboratorio de tablas ahora ocupa menos alto y se lee como tres tarjetas compactas en fila, con los datos internos distribuidos en columnas.

## Cambios clave

- Las tarjetas internas usan menos padding.
- El contenido de cada tarjeta se organiza en una grilla compacta.
- Las reglas siguen siendo las mismas, pero ahora entran mejor en pantalla.
- El encabezado del panel se mantiene en una sola fila y no compite con el contenido.

## Validaciones

- `npm run build`

## Notas

- Esta mejora es puramente visual y no cambia la logica del laboratorio.
