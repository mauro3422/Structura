# v1.2.10 - Reglas del laboratorio para tablas

## Resumen

Se agrego un panel de reglas simples para que el laboratorio explique su criterio de forma visible:

- Que es obligatorio.
- Que es sugerido.
- Que queda para una fase posterior.

## Detalles

- El laboratorio deja claro que la verdad del modelo vive en la FK y su destino.
- Se aclara que snake_case y minúsculas no son un error.
- Se muestra que 1:N sigue siendo una relacion valida y repetible en detalle.
- Se deja 1:1 y N:N como casos de mayor rigor o fase futura.

## Verificacion

- `npm run typecheck`
- `npm test`
- `npm run build`
