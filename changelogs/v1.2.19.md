# v1.2.19 - Reglas con ayuda expandible

_Fecha: 2026-04-22_

## Resumen

El panel de reglas del laboratorio ahora usa tarjetas expandibles. Cada regla muestra un resumen corto y al tocarla aparece una explicacion mas clara con ejemplos de lo que pide el laboratorio.

## Cambios clave

- Las reglas se presentan como acordeones tactiles.
- Cada tarjeta arranca compacta y muestra su ayuda al abrirla.
- La lectura en mobile mejora porque el usuario puede ver solo lo necesario.
- Se mantiene la separacion entre lo obligatorio, lo sugerido y la fase 2.

## Validaciones

- `npm run typecheck`
- `npm test`
- `npm run build`

## Notas

- Esta vuelta agrega un mecanismo didactico para complementar la teoria sin recargar la pantalla.
