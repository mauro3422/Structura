# v1.2.5 - Panel de relaciones para tablas

## Resumen

Esta iteracion agrega un panel explicito de relaciones dentro del laboratorio de tablas, para que la cardinalidad y los enlaces entre tablas se vean y se validen mejor.

## Cambios

- Se agrego un panel de relaciones que detecta FKs, tabla destino y cardinalidad.
- Se agrego inferencia de rol maestra/detalle para 1:N y una alerta de revision para 1:1 sin FK unica o compartida.
- Se agregaron estados visibles para relaciones correctamente enlazadas y relaciones incompletas.
- Se mantuvo el dibujo de conexiones en el canvas, pero ahora con una vista textual que acompana el modelo.
- Se sumo cobertura de tests para el panel de relaciones.

## Impacto

- Mejora la lectura pedagogica del laboratorio.
- Hace mas claro el paso de teoria a practica en 1:1 y 1:N.
- Permite detectar relaciones incompletas sin depender solo del grafico.
