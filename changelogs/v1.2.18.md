# v1.2.18 - Reglas apiladas en una sola columna

_Fecha: 2026-04-22_

## Resumen

El panel de reglas del laboratorio de tablas ahora apila las tres tarjetas una debajo de otra. Cada tarjeta usa el ancho completo disponible para que su contenido quede mas comodo y sin el efecto de columnas apretadas.

## Cambios clave

- Las tarjetas de reglas se muestran en una sola columna vertical.
- Cada tarjeta usa su propio ancho para acomodar mejor los chips internos.
- El panel conserva una lectura clara: obligatorias, sugeridas y fase 2.
- No cambio ninguna regla logica del laboratorio.

## Validaciones

- `npm run build`

## Notas

- Este ajuste es solo de maquetacion. La proxima iteracion puede enfocarse en afinar la densidad de cada tarjeta si hace falta.
