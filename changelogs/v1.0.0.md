# v1.0.0 - Structura Launch 🚀

**Fecha de Lanzamiento:** 15 de Abril de 2026

## Novedades Principales

- **Refactorización de "DataLab" a "Structura"**: Cambio de enfoque hacia arquitectura global de sistemas, diseño y lógica de software.
- **Página de Inicio Modular**: La *"Home"* principal ahora actúa como un catálogo de cursos/materias (`¿Qué querés aprender hoy?`), en vez de forzar a un único flujo de Estructura de Datos.
- **Redes de Computadoras**: 🌐 Agregado módulo completo de redes. Contiene conceptos sobre nodos, paquetes, infraestructura de internet, direcciones IP y mecanismos de funcionamiento de DNS.
- **UX/Gamificación "Juice"**:
  - `+` Se agregaron animaciones hápticas a nivel global (touchdown scales) en botones y tarjetas para una sensación 100% de APP nativa en móvil.
  - `+` Barra de progreso colorizada que detecta el llenado por módulo para visualizar lo que nos queda aprender.
  - `+` Checkmark (`✅`) inyectada en tiempo real cuando el sistema detecta vía memoria que una lección ha finalizado.
- **Progreso en Memoria**: Implementación de capa base `Progress.js` que utiliza el `localStorage` del móvil para emular el registro que un backend en Supabase tendrá pronto.
- **Estructura Estándar UI**: Corrección de nomenclaturas (Ej. Cambio estructural a botones base de "Siguiente Lección →").

## Optimizaciones de Sistema
- Extracción del sistema offline profundo / PWA estricto nativo para privilegiar un Development Server y Updates más veloces hacia Vercel (la app requerirá conexión a internet pero actualizará instantáneamente cuando el admin suba nuevas versiones al código fuente).
- Mejoras directas de DOM rendering doble (el motor ahora limpia e inyecta dinámicamente evitando el flasheo doble del router).
